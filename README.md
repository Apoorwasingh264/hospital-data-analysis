# 🏥 Hospital Data Analysis Project

## 📌 Project Overview
This project performs end-to-end analysis of hospital data to uncover insights related to patient behavior, appointment trends, revenue generation, and operational efficiency.

## 🛠 Tools Used
- SQL (Data Extraction & Analysis)
- Excel (Data Cleaning & Dashboard)
- Power BI (Interactive Dashboard)

## 🧠 SQL Concepts Used
- Joins (INNER, LEFT JOIN)
- Window Functions (RANK, ROW_NUMBER, LAG)
- Aggregations (SUM, COUNT, AVG)
- Date Functions (MONTH, YEAR)

## 📌 Business Problem
The hospital management wants to improve operational efficiency, reduce missed appointments, and optimize revenue generation using data-driven insights.

## 📊 Key Insights
- High cancellation and no-show rates indicate scheduling inefficiencies  
- Revenue is concentrated in a few medical specializations  
- A small number of patients contribute significantly to total revenue  
- Uneven doctor workload distribution may lead to resource imbalance  
- Payment failures and pending bills impact hospital cash flow  
- Patient demand shows seasonal variation  

## 📈 Dashboard Features
- Appointment Status Distribution
- Monthly Appointment Trends
- Revenue by Specialization
- Doctor Workload Analysis
- Payment Status Insights
